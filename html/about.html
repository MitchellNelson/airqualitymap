Tools and Features

  • Two side by side maps using Leaflet
  • Vue js was used to created the responsive nature of the page
  • Gets air quality measurements from the Open AQ Platform for the past 30 days for the area within view of the maps
  • UI Filters to choose air particles of interest
  • Heatmap visualization overlay on the map when only one particle type selected
  • Background color of particle values in the table match the Air Quality Index from the EPA
  • Uses the Nominatim API to search via place name in addition to lat/long coordinates

Interesting Finds

  • Despite its large size, Portland has almost all good air quality indexes
  • We expected Yellowstone to have high amounts of so2 but the level are between 0-1, where we found measurements
  • Black Carbon is supported by OpenAQ but is measured in very few cities
  • We found many cities such as San Antonio have good conditions during the night but moderate quality during the day
  • New Delhi has the only hazardous level of pm2.5 that we have found
  • Almost all measurements in Bejing record all of the parameters (excluding black carbon) which is not true for most other places
